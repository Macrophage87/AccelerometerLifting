# AccelerometerLifting

Android app for velocity-based strength training using a WitMotion WT901BLECL
BLE accelerometer (bar speed + rep phase/tempo tracking) and an optional
Garmin HRM 600 heart rate monitor, with LLM-friendly plan import and session
export for analysis in Claude.

**Status: specification phase.** No app code yet — see [`PROMPT.md`](PROMPT.md)
for the full engineered build prompt / technical specification, including BLE
protocol details, signal-processing requirements, data schemas, architecture,
CI/CD design, and implementation milestones.
